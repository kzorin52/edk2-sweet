## Building

1. Clone this project

```bash
git clone https://github.com/kzorin52/edk2-sweet.git --depth=1
cd edk2-sweet
```

2. Build this project (only on linux)

```bash
./build_common.sh
./sweetbuild.sh
```

## Available Function  
  
 * USB (need extra power supply)  
 * UFS
